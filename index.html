Memory Game

This game will test your memory skills and is a classic play on the old version of the card game.

How to Play:-

  • There will be 16 cards in the game. Each card will have a number written on it.
  • There will be 8 pair of total cards.
  • Your job is to select the Cards pair-wise before the number of cards run out.
  • You will Get 8 Guesses.
